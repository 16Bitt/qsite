# No AI Policy

This repository has been authored without the assistance of an LLM or any other form of AI-assisted
output. As a courtesy to the original author, please refrain from the use of any AI assistance when
opening issues, pull requests or making any other contribution to this repository.

While there are more complex reasons that could warrant tomes of text, in short:

- This project was authored by hand because the author finds value in writing software
- There is no economic incentive for this project, thus there is no point in trading authorship for
  code volume
- The author prefers the concise, albeit sometimes grammatically incorrect, correspondence of humans
  over the verbose and bloated output of LLMs
- The nature of copyright, intellectual property, and ethics is at best ambiguous when it comes to
  using LLM output

Any suspected use of LLMs will result in the immediate closure of the contribution to this project.
